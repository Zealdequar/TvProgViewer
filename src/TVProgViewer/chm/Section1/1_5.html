Основное меню приложения TVProgViewer

Основное меню содержит команды для управления приложением и состоит из пунктов: «Файл», «Вид», «Фильтры», «Настройка» и «Справка», а также их подпунктов:

  • «Загрузить» — осуществляет загрузку телепрограммы из интернета.
  • «Поиск» — осуществляет отображения диалога для ввода нужных критериев и поиска телепередач.
  • «Выход» — осуществляет остановку работы приложения и освобождения выделенной под его работу памяти.
  • «Сейчас», «Затем», «По дням», «По каналам» — режимы отображения телепрограммы.
  • «Анонсы»: «Если присутствуют», «Всегда», «Никогда» — режимы отображения анонсов к передачам.
  • «Показать/скрыть»: «Главное меню», «Панель инструментов», «Строку состояния» — команды для определения показывать или не показывать соотоветсвующие элементы управления.
  • «Фильтры»: «По жанрам», «По рейтингам», «По анонсам», «По напоминаниям» — режимы фильтрации телепередач.
  • «Текущие напоминания» — отображение окна с запланированными напопоминаниями.
  • «Настройка»: «Классификация (типы) передач» — открывает редактор жанров, «Настройка рейтингов» — открывает редактор рейтингов, «Настройка каналов» — открывает редактор каналов, «Общие настроки» — открывает общие для приложения настройки.
  • «Справка» — открывает настоящее руководство
  • «Регистрация» — открывает окно для добровольной регистрации пользователя.
  • «О программе» — открывает окно с данными о версии приложения, разработчике, а также дополнительной информации.

Некоторые команды основного меню продублированы в панели инструментов.
